## Mobile Responsive Patterns Applied

### Navbar
- Responsive text sizes: `text-xl md:text-2xl`
- Hidden items on mobile: `hidden md:inline`
- Flexible spacing: `space-x-4 md:space-x-8`
- Wrapping: `flex-wrap gap-4`

### Typography
- Responsive headings: `text-2xl md:text-3xl md:text-4xl`
- Mobile-friendly font sizes throughout
- Proper line-height for readability

### Grids & Layouts
- Product grids: `grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4`
- Responsive gaps: `gap-6 md:gap-8`
- Flexible containers: `flex-col md:flex-row`

### Tables
- Desktop: Full table view
- Mobile: Card-based layout with `hidden md:block` and `md:hidden`

### Forms
- Full-width inputs on mobile
- Responsive padding: `p-4 md:p-6 md:p-8`
- Touch-friendly button sizes

### Images
- Responsive heights: `h-48 md:h-56 md:h-64`
- Proper object-fit for all screen sizes
